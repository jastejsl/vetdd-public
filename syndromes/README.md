# Syndromes

In this section, you will find all the various syndromes that occur in veterinary science.

To create a new entry, please copy the template provided under "Sydrome Template", head over to (insert link), click "edit" and fill in the informaiton to be added.

To edit the information in an already existing entry on VetDD, open the "syndrome-files" dropdownm, chose the disease to edit, click the "Edit" button in the top right corner and edit the fields as needed.
